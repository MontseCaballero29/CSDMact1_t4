## Descripción del proyecto

**Manos de Oaxaca** es una aplicación REST desarrollada con Java y Spring Boot para consultar información relacionada con talleres artesanales del estado de Oaxaca.

El proyecto permite identificar la aplicación que se encuentra ejecutándose, consultar una lista de talleres obtenida desde una API externa y mostrar el nombre y la ubicación de uno de los talleres disponibles.

Esta aplicación fue desarrollada como parte de la **Actividad 1 del Tema 4** de la materia de **Programación Web**, con el propósito de configurar el entorno de desarrollo para Java y Spring Boot, crear controladores REST, implementar endpoints y preparar el proyecto para su despliegue en un VPS.
